# Statistical Analysis in Excel

Two projects: correlation/regression analysis and histogram/descriptive statistics.

## Files

- [correlation-regression.xlsx](./correlation-regression.xlsx) - Correlation and regression analysis
- [histogram-statistics.xlsx](./histogram-statistics.xlsx) - Frequency distribution, histogram, descriptive statistics

---

## correlation-regression.xlsx

**Task**: Analyze the relationship between two variables. Generate 200 data points, calculate correlation coefficient, build regression models.

**What I did**:
- Generated random data (normal distribution, uniform distribution)
- Calculated Pearson correlation coefficient (3 methods: Analysis ToolPak, CORREL function, manual formula)
- Built scatter plots
- Performed regression analysis using:
  - Regression module (Analysis ToolPak)
  - LINEST function
  - Manual formulas (normal equations)
- Compared results across all methods

---

## histogram-statistics.xlsx

**Task**: Generate random numbers, build frequency distribution and histogram, calculate descriptive statistics.

**What I did**:
- Generated 100 random numbers (normal distribution: mean=0, sd=1)
- Generated 100 random numbers (uniform distribution: 0 to 1)
- Defined bins (intervals) for both distributions
- Counted frequencies using `FREQUENCY` function
- Built histograms for both distributions
- Calculated descriptive statistics (mean, median, min, max, etc.)

# Статистический анализ в Excel

Два проекта: корреляционно-регрессионный анализ и гистограммы с описательной статистикой.

## Файлы

- [correlation-regression.xlsx](./correlation-regression.xlsx) - Корреляционный и регрессионный анализ
- [histogram-statistics.xlsx](./histogram-statistics.xlsx) - Распределение частот, гистограмма, описательная статистика

---

## correlation-regression.xlsx

**Задача**: Проанализировать взаимосвязь между двумя переменными. Сгенерировать 200 точек, рассчитать коэффициент корреляции, построить модели регрессии.

**Что сделано**:
- Сгенерированы случайные данные (нормальное и равномерное распределение)
- Рассчитан коэффициент корреляции Пирсона (3 способа: Пакет анализа, функция КОРРЕЛ, ручная формула)
- Построены диаграммы разброса
- Выполнен регрессионный анализ через:
  - Пакет анализа (Регрессия)
  - Функцию ЛИНЕЙН
  - Решение системы нормальных уравнений (ручной расчёт)
- Сравнены результаты всех методов

---

## histogram-statistics.xlsx

**Задача**: Сгенерировать случайные числа, построить распределение частот и гистограмму, рассчитать описательную статистику.

**Что сделано**:
- Сгенерировано 100 случайных чисел (нормальное распределение: среднее=0, ст.откл=1)
- Сгенерировано 100 случайных чисел (равномерное распределение: от 0 до 1)
- Заданы интервалы (карманы) для обоих распределений
- Подсчитаны частоты через функцию ЧАСТОТА
- Построены гистограммы для обоих распределений
- Рассчитана описательная статистика (среднее, медиана, мин, макс и т.д.)
