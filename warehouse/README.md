# Warehouse Accounting

This file contains a warehouse accounting system built in Excel.

## What's inside

- **Directories**: Suppliers, customers, warehouse locations, product catalog
- **Income sheet**: Recording incoming goods with dropdown lists (product names, suppliers, locations)
- **Expense sheet**: Recording outgoing goods with dropdown lists
- **Turnover report**: Summary of income, expense, and current balances

## How it works

1. User selects a product from a dropdown list
2. Unit of measure and price are filled automatically (using VLOOKUP)
3. User enters quantity and price → total cost is calculated
4. Data is saved to Income or Expense sheet
5. Turnover report shows balances for each product

## Skills demonstrated

- Excel tables and named ranges
- Data validation (dropdown lists)
- VLOOKUP and SUMIFS functions
- Error handling (IFERROR / ISNA)
- Inventory tracking (income, expense, balance)

# Складской учёт

Этот файл содержит систему складского учёта, построенную в Excel.

## Что внутри

- **Справочники**: Поставщики, покупатели, точки учёта, номенклатура товаров
- **Лист «Приход»**: Учёт поступления товаров с выпадающими списками
- **Лист «Расход»**: Учёт отгрузки товаров с выпадающими списками
- **Лист «Оборотная ведомость»**: Сводка по приходу, расходу и текущим остаткам

## Как работает

1. Пользователь выбирает товар из выпадающего списка
2. Единица измерения и цена подставляются автоматически (через ВПР)
3. Пользователь вводит количество и цену → стоимость считается
4. Данные сохраняются в лист «Приход» или «Расход»
5. В оборотной ведомости видны остатки по каждому товару

## Навыки

- Таблицы Excel и именованные диапазоны
- Проверка данных (выпадающие списки)
- Функции ВПР и СУММЕСЛИМН
- Обработка ошибок (ЕСЛИОШИБКА / ЕНД)
- Складской учёт (приход, расход, остатки)
